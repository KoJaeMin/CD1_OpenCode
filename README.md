# CD1_OpenCode
2022년 1학기 컴퓨터공학과 캡스톤디자인1 오픈코드팀

## 주의 사항

### git pull

무조건 pull하신 후에 작업하시기를 바랍니다.

```bash
    git pull
```

### 가상환경 구축 및 실행

```bash
    python3 -m venv env
    source ./env/bin/activate
```

### 필요한 library 설치

```bash
    pip install -r requirements.txt
```

### 추가 library 설치 후 equirement.txt에 추가

```bash
    pip freeze > requirements.txt
```

### virtual enviroment 종료

```bash
    deactivate
```
